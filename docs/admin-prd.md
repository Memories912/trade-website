# 后台管理系统 PRD

## 项目信息

- **Language**: 中文
- **Programming Language**: Vue 3 + Vite + Tailwind CSS
- **Project Name**: trade_website_admin
- **原始需求**: 为外贸 B2B 交易网站设计后台管理系统，嵌入现有项目 `/admin` 路由下，使用 Mock 数据，覆盖仪表盘、用户管理、产品管理、询盘管理、采购寻源管理、知识库管理、AI 聊天记录、系统设置等功能模块。

---

## 产品定义

### Product Goals

1. **运营效率提升**：管理员可通过后台一站式管理产品、询盘、寻源请求等核心业务数据，减少人工操作成本
2. **业务洞察可视化**：通过仪表盘关键指标与趋势图表，帮助管理者快速掌握业务运行状况并做出决策
3. **多语言内容管控**：支撑中/英/韩/日四语内容的统一录入与管理，确保用户端多语言展示的一致性与准确性

### User Stories

1. 作为管理员，我希望登录后台后看到关键业务指标（询盘数、用户数、产品数等），以便快速了解平台运营状况
2. 作为管理员，我希望对产品进行增删改查并管理分类和认证标签，以便及时更新产品信息
3. 作为管理员，我希望查看和处理询盘并跟踪其状态流转，以便高效响应客户需求
4. 作为管理员，我希望管理采购寻源请求并提交报价，以便促成交易
5. 作为管理员，我希望在同一界面编辑知识库 FAQ 的多语言内容，以便减少切换成本

### 配色与风格

- 与用户端一致：主色 `#fd560f`，深色 `#282828`，背景 `#f6f6f8`
- 侧边栏深色背景（`#282828`），白色文字，当前项用主色高亮
- 内容区浅色背景（`#f6f6f8`），白色卡片，圆角 + 阴影
- 表格行 hover 浅灰高亮，操作按钮主色/文字色
- 字体：Poppins + 系统字体

### 各页面设计

#### 1. 登录页

- 居中白色登录卡片，左侧/背景品牌视觉
- 字段：邮箱 + 密码，登录按钮（主色）
- Mock 账号：admin@trade.com / admin123

#### 2. 仪表盘（Dashboard）

- 顶部 4 个统计卡片：总用户数、总产品数、待处理询盘、待处理寻源（图标+数字+环比趋势箭头）
- 中部：近期询盘趋势折线图（近 30 天）
- 下部左右分栏：
  - 左：最新询盘列表（最近 5 条，含产品名/客户/状态/时间）
  - 右：最新寻源列表（最近 5 条，含标题/客户/状态/时间）

#### 3. 用户管理

- 列表页：搜索框 + 国家筛选 + 状态筛选 + 表格（头像/名称/邮箱/公司/国家/状态/注册时间/操作）
- 操作：查看详情、启用/禁用
- 详情弹窗/抽屉：用户完整信息 + 历史询盘/寻源/收藏列表

#### 4. 产品管理

- **产品列表**：搜索 + 分类筛选 + 表格（图片/名称/分类/MOQ/状态/操作）+ 分页
- **新增/编辑产品**：表单页或抽屉
  - 多语言 Tab 切换（中/英/韩/日）编辑 name、description
  - 通用字段：category（下拉）、MOQ、lead_time、packaging、features（动态添加）
  - 图片上传区：主图 + 多图
  - 认证选择：多选 checkbox
  - 规格编辑：Key-Value 动态行
- **分类管理**：简单表格 + 内联编辑，支持拖拽排序
- **认证管理**：简单表格 + 弹窗编辑（code, name, icon）

#### 5. 询盘管理

- 列表页：搜索 + 状态筛选（pending/processing/quoted/closed）+ 日期范围 + 表格（编号/产品/客户/状态/时间/操作）
- 详情页/抽屉：
  - 客户信息 + 产品信息 + 询盘内容
  - 状态流转按钮（操作区）
  - 内部备注（时间线形式，可添加）
  - 回复区（P1：文本框 + 发送按钮）

#### 6. 采购寻源管理

- 列表页：搜索 + 状态筛选 + 表格（编号/标题/客户/状态/时间/操作）
- 详情页/抽屉：
  - 客户信息 + 寻源需求描述
  - 状态流转按钮
  - 报价区：填写报价金额、交期、备注（仅 quoted 状态后可填）

#### 7. 知识库管理

- 列表页：搜索 + 标签筛选 + 表格（问题/标签/操作）
- 新增/编辑：多语言 Tab 切换编辑 q/a，标签多选

#### 8. AI 聊天记录

- 左侧用户列表（按最近聊天时间排序，搜索框）
- 右侧聊天记录时间线（气泡样式，按时间排列）

#### 9. 系统设置

- Tab 切换：公司信息 / 语言配置
- 公司信息：表单编辑，多语言 Tab 切换公司名称
- 语言配置：启用的语言列表，开关切换，默认语言选择

## Open Questions

1. **权限体系**：当前是否需要角色权限（如超级管理员/运营人员），还是单一管理员角色即可？
2. **数据删除策略**：产品和用户删除是软删除还是硬删除？是否需要回收站功能？
3. **询盘回复方式**：询盘回复是纯站内备注，还是需要模拟邮件发送？
4. **图片存储**：产品图片上传的 Mock 方案——使用本地 base64 还是 placeholder 图片服务？
5. **移动端适配**：后台是否需要完整的移动端适配，还是仅保证平板可用即可？
6. **路由守卫**：管理员与普通用户是否共享同一用户体系？管理员登录后是否同时能看到用户端和管理端？
